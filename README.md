# Lokender Pratap Portfolio

A personal portfolio website for Lokender Pratap, a final-year B.Tech Information Technology student focusing on Data Analytics, Machine Learning, AI Engineering, and Full-Stack Development.

## Project Overview

This project is a static portfolio website built with HTML, CSS, and JavaScript. It showcases Lokender's professional profile, technical skills, featured projects, achievements, GitHub activity, and contact information.

## Features

- Responsive portfolio layout with a dark/light theme toggle
- Animated entry reveal effects for page sections
- Global visitor counter using CountAPI
- Interactive project filter buttons
- GitHub activity contribution graph rendering
- Mobile-friendly navigation with accessible skip link
- Back-to-top button for easy navigation
- External contact / social links

## Technology Stack

- HTML
- CSS
- JavaScript

## Project Structure

- `index.html` — main portfolio page
- `styles.css` — site styling, layout, responsive rules, and animations
- `script.js` — theme handling, navigation toggling, animations, visitor count, project filtering, GitHub contributions, and back-to-top behavior
- `profilr-picture.jpeg` — profile image used in the hero section
- `resume-lokender-pratap.pdf` — downloadable resume

## Usage

1. Open `index.html` in your browser.
2. Use the navigation links or scroll to explore sections like About, Skills, Projects, GitHub, and Contact.
3. Toggle the theme using the Theme button in the header.
4. Filter project cards using the filter buttons in the Projects section.

## Deployment

- Deployed on Vercel as a public static website.
- Live URL: https://lokender-portfolio-tan.vercel.app
- Any code changes must be redeployed to update the live site.

## Notes

- The GitHub contribution graph is generated by fetching public GitHub events and falls back to a static pattern on API failure.
- Visitor count is now powered by CountAPI for global visit tracking, with a local fallback when the API is unavailable.
